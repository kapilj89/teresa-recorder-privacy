Privacy Policy — Teresa Recorder

Last Updated:14 December 2025

1. Introduction

Teresa Recorder (“the Extension”) helps users record web interactions for automated test generation. We respect user privacy and strive to ensure transparency in data handling. This Privacy Policy describes what data is collected and how it is used.

2. Information We Collect

2.1 Personal Data

The Extension does not collect, store, transmit, or share personally identifiable information (PII).

2.2 Browsing Data

The Extension may momentarily access webpage elements (DOM structure, attributes, selectors) only when the user enables recording. This data is:

  • Processed locally in the user’s browser
  • Used solely to generate automation steps
  • Never transmitted or stored externally

2.3 Technical Data

The Extension does not collect IP addresses, device identifiers, cookies, or analytics.

3. How We Use the Information

Any accessed information is used strictly for functionality:

  • Generating automation instructions
  • Identifying elements during recording

All processing occurs locally on the device.

4. Permissions Explained

  • activeTab: Allows access to the active tab only when initiated by the user.
  • scripting / tabs: Enables capturing interactions and DOM details.
  • storage: Stores extension preferences locally.

No permission collects personal or sensitive data.

5. Data Storage & Retention

The Extension does not store any browsing or personal data externally.

6. Data Sharing

No data is shared, sold, or transmitted to third parties.

7. Children’s Privacy

The Extension is not intended for children under 13. No data is knowingly collected from children.

8. Security

All processing occurs locally, ensuring that no unauthorized access or logging of user data occurs.

9. Changes to this Privacy Policy

We may update this Privacy Policy to reflect feature changes. Continued use indicates acceptance of updates.
